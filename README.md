Install my-project/frontend with npm

1. To install
```bash
  git clone https://github.com/Santhosh1507/food-hotel-mern.git/
```
2. frontend project
```bash
  cd .\food-hotel\frontend\
```
3. install package
```bash

  npm run dev
```
